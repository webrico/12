1) установить nodejs
2) через него npm install -g --save  возможно продублировать в папку с проектом без "-g"
3) обновить пакеты если нужно npm update --save
4) установить гит git init
5) в package.json поменять имя проэкта.  там же внизу настроить сетку.
6) пересобрать файл сетки  gulp smartgrid. проверить подключён ли он в general.less

стандартные команды:

gulp'a: 
gulp - запускает сервеh и следит за изменениями less и html
gulp smartgrid - собирает сетку и бросает её в dev/less/import/smart-grid.less
gulp srite - собирате спрайт из .png картинок добавленных в dev/img/icons/   сам спрайт в img/sprite.png
gulp move - перемещает шрифты картинки и js из рабочей папки dev  и готовую build
gulp css - собирает less в css
gulp html - собирает все части html в index


git'a:
git init - говорит гиту следить за этой папкой.
git add . - добавляет изменённые папки\файлы
git status - показывает какие файлы изменились
git commit -m 'любой коментарий' - делает "слепок" текущей разработки и помечает его коментарием.
git pull
git push
